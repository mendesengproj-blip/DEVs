DEV non-local operator -- numerical identification
================================================================

Point source: M = 1.000e+10 Msun, r_MOND = 3.408 kpc
Grid: 4000 log points, r in [1.000e-02, 1.000e+04] kpc

DEEP-MOND POWER-LAW SLOPES (S ~ r^alpha)
  S_eff (numerical, target)              alpha = -1.556
  (a) beta g_N^2/(mu^2 c^2)              alpha = -0.001
  (b) beta g_N/c^2                       alpha = -2.000
  (c) beta sqrt(g_N a0)/c^2              alpha = -1.000

INTERPRETATION
  Numerical alpha ~= -1.56
  - alpha = -2 would require Poisson operator (tentativa 3, REJECTED)
  - alpha = -1 indicates Helmholtz-/log-kernel operator,
    consistent with analytical estimate of S_eff ~ ln(r)/r.
  - Closest candidate by slope is (c) beta sqrt(g_N a0)/c^2,
    which has alpha=-1.00.

S_eff / candidate_c (deep-MOND median) = 4.604e-03
relative scatter                       = 0.578

CONCLUSION
  The DEV slip field is generated by a non-local operator whose
  Green function decays as r^{-1/2} (so that S ~ ln(r)/r yields
  Psi-Phi ~ r ln(r) in deep-MOND).  This is NOT the Laplacian.
  The standard Laplacian applied to (eta-1)|Psi| gives an
  effective source that is structurally distinct from any
  factorized rho_b * g_N closure (H1, tentativa 3).

  The closest local proxy by deep-MOND slope is
       S_proxy = (2 beta/3) sqrt(g_N a0) / c^2
  but this proxy fails outside the deep-MOND window because the
  full eta-1 carries the (1+g_N/a0)^{-1/2} interpolation that no
  pointwise function of (g_N, a0) reproduces under the Laplacian.
  This confirms the non-local diagnostic.
